# Project Name

Triangle Brain is a minimalist puzzle game designed to challenge your logic,
pattern recognition, and strategic thinking.

• Simple rules, deep gameplay
• Classic and Zen modes
• Daily challenges to keep your mind sharp
• Clean visuals with smooth animations
• Play offline, anytime

Perfect for quick breaks or long thinking sessions.
Train your brain — one triangle at a time.


## Tech Stack
- Kotlin
- Android SDK
- MVVM Architecture
- Retrofit
- Room Database

## Installation

### Clone the repository
https://github.com/kennisolu/TriangleBrainPuzzle.git

### Open in Android Studio
1. Open Android Studio
2. Select "Open Project"
3. Choose the cloned folder

### Build project
Run the app using emulator or device.
